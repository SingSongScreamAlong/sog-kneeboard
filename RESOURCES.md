## 🗺️ Map Extraction Tools

### GRAD Map Export Helper
- **GitHub**: https://github.com/gruppe-adler/grad_meh
- **Purpose**: Export Arma 3 maps to web tiles
- **Installation**: `npm install -g grad-meh`
- **Usage**: `grad-meh --world vn_cam_lao_nam --output ./tiles`

### Arma 3 Map Exporter (Python)
- **GitHub**: https://github.com/veteran29/arma3-map-exporter
- **Purpose**: Python-based map extraction
- **Installation**: `pip install arma3-map-exporter`

### A3ME (Arma 3 Map Exporter)
- **GitHub**: https://github.com/gruppe-adler/a3me
- **Purpose**: Web-based map viewer and exporter
- **Demo**: https://gruppe-adler.de/map/

## 🎨 Map Assets & Tiles

### S.O.G. Prairie Fire Maps

#### Cam Lao Nam
- **Size**: 5120x5120 meters
- **Type**: Main operational area
- **Features**: Jungle, rivers, villages, airfields
- **World Name**: `vn_cam_lao_nam`

#### Khe Sanh
- **Size**: 2048x2048 meters
- **Type**: Combat base
- **Features**: Airstrip, bunkers, firebase
- **World Name**: `vn_khe_sanh`

#### The Bra
- **Size**: 1024x1024 meters
- **Type**: Small operations area
- **Features**: Dense jungle, river crossing
- **World Name**: `vn_the_bra`

### Tile Sources

**Note**: Map tiles must be extracted from your legal copy of Arma 3.

#### Extraction Methods:
1. **PBO Extraction** (see tools/extract-map-tiles.md)
2. **GRAD Map Export** (automated)
3. **In-game Screenshots** (manual)

- Check Arma 3 modding Discord for shared resources
