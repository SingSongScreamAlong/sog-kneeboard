# BroadcastBox Keyboard Shortcuts

## Driver Selection
| Key | Action |
|-----|--------|
| `1` | Select driver 1 from stack |
| `2` | Select driver 2 from stack |
| `3` | Select driver 3 from stack |
| `4` | Select driver 4 from stack |
| `5` | Select driver 5 from stack |

## Camera Controls
| Key | Action |
|-----|--------|
| `Space` | Lock/unlock current camera |
| `Tab` | Cycle camera mode (world → onboard → battle) |
| `←` | Previous AI suggestion |
| `→` | Next AI suggestion |
| `Enter` | Accept current AI suggestion |

## Replay
| Key | Action |
|-----|--------|
| `R` | Replay last bookmarked event |

## UI Controls
| Key | Action |
|-----|--------|
| `L` | Toggle leaderboard expansion |
| `A` | Toggle advanced options panel |
| `Esc` | Clear selection / exit mode |

## Notes

- Keyboard shortcuts are disabled when focus is in an input field
- Camera lock prevents AI from auto-switching cameras
- AI suggestions show confidence scores (higher = more relevant)
- Human override always takes priority over AI suggestions
