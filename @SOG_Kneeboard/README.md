# SOG Kneeboard Integration Mod

Real-time tactical kneeboard integration for Arma 3 S.O.G. Prairie Fire DLC.

## Installation

1. Copy this `@SOG_Kneeboard` folder to your Arma 3 directory
2. Enable the mod in Arma 3 launcher  
3. Start the SOG Kneeboard server on your PC
4. Launch Arma 3 with S.O.G. Prairie Fire

## Features

- **Real-time position tracking** - Syncs your player position and heading
- **Automatic map detection** - Recognizes Vietnam maps  
- **Bidirectional marker sync** - Markers from kneeboard appear in-game
- **Client-side only** - Works on any server

## Usage

The mod runs automatically when you join a server or start single-player. You'll see chat messages indicating the integration status.

**Chat Messages:**
- "SOG Kneeboard: Integration initialized"
- "SOG Kneeboard: Position tracking ACTIVE"  
- "SOG Kneeboard: X new marker(s) added to map"

## Requirements

- Arma 3 with S.O.G. Prairie Fire DLC
- SOG Kneeboard server running on same PC
